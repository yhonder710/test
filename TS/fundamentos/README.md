# Fundamentos de TypeScript

Este directorio contiene una colección de módulos que cubren los fundamentos de TypeScript de forma progresiva y práctica.

Cómo usar:
- Instala dependencias del proyecto raíz (ya existente): `npm install`
- Ejecuta ejemplos con ts-node desde la raíz o compílalos con `npm run build` y ejecuta los archivos resultantes en dist/.
- Para probar un ejemplo individual, usa: `npx ts-node fundamentos/01_basics.ts` (o el archivo que quieras).

Organización:
- 01_basics.ts: fundamentos básicos de tipos y estructuras simples
- 02_types_interfaces.ts: interfaces y aliases
- 03_classes.ts: clases y herencia
- 04_generics.ts: generics
- 05_unions_intersections.ts: uniones e intersecciones
- 06_advanced_types.ts: tipos avanzados y utilitarios
- 07_functions.ts: funciones y tipado de funciones
- 08_modules.ts y 09_modules_app.ts: módulos y ejemplos de import/export
- 09_decorators.ts: decoradores (experimental)
- 10_async.ts: async/await
- 11_exercises.md: ejercicios propuestos
