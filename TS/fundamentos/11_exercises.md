# Ejercicios de Fundamentos TS

- [ ] Escribe una función tipada que acepte un array de números y retorne la suma.
- [ ] Define una interfaz para un Producto con id, nombre y precio; crea una función que imprime un producto.
- [ ] Crea una clase Animal con método speak; extiéndelo a Perro y Gato con implementaciones.
- [ ] Usa generics para crear una función que invierta una tupla de dos elementos de cualquier tipo.
- [ ] Explora tipos avanzados: Optional, Partial, Readonly y Record con ejemplos simples.
